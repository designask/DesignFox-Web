# DesignFox - Creative Digital Agency Website

A modern, responsive one-page website for **DesignFox** creative digital agency.

## 🦊 About

DesignFox is a full-service creative digital agency specializing in:
- 🎨 Graphic Design
- 🌐 Web Design
- 💻 Software Development
- 📢 Digital Marketing

## 🚀 Features

- Modern one-page design inspired by premium agency themes
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations
- Animated counter section
- Interactive portfolio gallery
- Contact form with validation
- Mobile-friendly navigation
- Scroll-to-top button
- CSS gradient backgrounds & animations

## 📁 Project Structure

```
DesignFox-Web/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # JavaScript functionality
└── README.md           # Project documentation
```

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Poppins)

## 🌐 Hosting

This site is designed to be hosted on **GitHub Pages**:
1. Go to repository Settings
2. Navigate to Pages section
3. Select `main` branch as source
4. Save and your site will be live!
